# Terra Concordia · Public

This repository hosts public-facing materials for **Terra Concordia: Civilizations in Harmony** — a digital eurogame by NCB Company.

> ⚠️ **The game source code is NOT public.** This repo contains only press kit, release notes, achievement icons, capsule arts, and community contribution guides.

---

## 🎮 About the Game

Terra Concordia is a deep, fully-featured digital eurogame for **1-4 players**, releasing on Steam. Build civilizations, research technologies, erect Wonders — but cooperate to keep the world's Global Sustainability above zero, or everyone loses.

- 🌐 **Website:** https://ncbgames.com
- 🎮 **Steam:** https://store.steampowered.com/app/4791670/
- 💬 **Discord:** [Join our community](https://discord.gg/Xp869eSNWP)
- 📰 **Press Kit:** [press-kit/](press-kit/)
- 📖 **Manuals:** [Rulebook, Solo guide & Print & Play](https://ncbgames.com/manuals-en.html)
- 🎬 **Trailer:** Coming soon
- 📧 **Contact:** terraconcordia@ncbgames.com
