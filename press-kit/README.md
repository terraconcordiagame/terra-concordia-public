# Press Kit

Material para imprensa, influencers e parceiros.

## Estrutura

```
press-kit/
├── factsheet.md         # Ficha técnica do jogo (será gerado na Fase 3)
├── description.md       # Descrições curta/média/longa em PT/EN
├── contact.md           # Como contatar a equipe
├── logos/               # Logos em PNG/SVG, fundo transparente, várias resoluções
├── screenshots/         # Mesmas selecionadas da Steam (6-10 melhores)
├── team-bios/           # Bio NCB Company (em PT e EN)
└── press-quotes/        # Vazio inicialmente; preencher após reviews
```

## Quando enviar

Press kit deve ser enviado para imprensa:
- **T-30 dias** (30 dias antes do lançamento): primeiro envio
- **T-7 dias**: lembrete + key de acesso antecipado
- **T-0**: confirmação no dia do lançamento
