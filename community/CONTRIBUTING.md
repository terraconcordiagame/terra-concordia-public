# Contributing to Terra Concordia

Thank you for considering contributing! Even though the game source is closed, there are many ways to help.

---

## 🌐 Translation Contributions

We currently support:
- 🇧🇷 Portuguese (Brazil) — 100% complete
- 🇺🇸 English — 100% complete
- 🇪🇸 Spanish — 70% complete (community help welcome)

We'd love to add:
- 🇨🇳 Chinese Simplified (huge market)
- 🇫🇷 French
- 🇩🇪 German
- 🇷🇺 Russian
- 🇯🇵 Japanese

### How to help translate

1. Join Discord #localization channel
2. Pick a language not yet covered
3. We send you a CSV with PT + EN strings (142 cards + UI)
4. Translate as PR or send back as CSV
5. Get credit in-game + Discord cargo "🌐 Translator"

## 🐛 Bug Reports

Two ways:

### Quick: Discord
Discord #bug-reports — use template fixed in topic

### Detailed: GitHub Issues
[Open new issue](../../issues/new) using our template:

```markdown
**Versão:** v1.0.X
**OS:** Windows 10/11
**Modo:** Solo / Standard / Diplomacy

**What happened:**

**Expected:**

**Steps to reproduce:**
1.
2.
3.

**Frequency:**
- [ ] Always
- [ ] Sometimes
- [ ] Once

**Screenshot/video:**
(optional)

**Save file:**
%APPDATA%/terra-concordia/saves/
(if relevant, attach JSON)
```

## 💡 Feature Suggestions

[Open discussion](../../discussions/new) using template:

```markdown
**Summary:**
(One-liner)

**Problem:**
What problem does this solve?

**Proposed solution:**
How should it work?

**Use cases:**
When would this help? Who benefits?

**Alternatives considered:**
Other ways to solve this?
```

NCB reviews all suggestions monthly. Most popular ones become features.

## 💬 Code Contributions

Game source is **closed**. We can't accept code PRs to the game itself.

However, you CAN contribute to:

### Open source tools
- **Discord bot** ([discord-bot/](https://github.com/ncbgames/terra-concordia-discord-bot)) — MIT license
- **Press kit website** (this repo) — assets CC BY-NC 4.0
- **Community Wiki** (separate repo) — when established

### Workshop content (future)
After Workshop integration in v2.X (post-launch):
- Custom scenarios
- Custom decks
- Cosmetic civilizations
- Tutorials

## 📜 Code of Conduct

Be kind. Be respectful. Be constructive.

### Not tolerated
- ❌ Racism, sexism, homophobia, xenophobia
- ❌ Personal attacks
- ❌ Spam or self-promotion
- ❌ Doxxing
- ❌ Harassment

### Consequences
- 1st violation: warning
- 2nd violation: 24h timeout
- 3rd violation: permanent ban
- Serious violations (threats, doxxing): immediate ban
